# Créez une application web de location immobilière avec React

![kasa logo](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTZ6hFqAQXkSHoUb0eSoUTOcE6X5RSJWW52jw&s)

Le projet 6 de la formation "Développeur Web" d'OpenClassrooms était de réaliser un site de location immobilière (du nom de Kasa), tout en utilisant la librairie React, et des outils comme Sass pour le CSS de l'application.

**Les objectifs de ce projet :**

_Initialiser une application avec Create React App_\
_Configurer la navigation entre les pages de l'application avec React Router_\
_Développer des éléments de l'interface d'un site web grâce à des composants React_\
_Mettre en œuvre des animations CSS_\
_Développer une interface web avec Sass_
